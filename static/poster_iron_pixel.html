Core Platform Features

Everything an institution needs, built in from day one.

🎓
Multi-Step Online Applications
A seamless 7-stage application journey that guides prospective students from programme selection to document submission — all in-browser, no paperwork.
  • Programme & qualification selector
  • Personal info & document upload
  • Auto-generated application reference
  • Draft save & resume at any step
  • Confirmation email on submit
🤖
AI-Powered Application Screening
Every application is assessed instantly by an AI screening engine (GPT-4o-mini) that scores, ranks, and flags strengths & concerns for admins.
  • Numeric screening score (0–100)
  • AI recommendation: Accept / Waitlist / Reject
  • Strengths, concerns & missing items (JSONB)
  • Rules-only fallback when AI is offline
  • Full screening audit trail per application
🏫
School & Curriculum Management
Manage an entire institution hierarchy — from the top-level school profile down to departments, courses, modules, subjects, and timetables.
  • Multi-school support (Pre / Primary / High / Tertiary)
  • Departments → Courses → Modules chain
  • Curriculum builder with drag-and-drop
  • School logo & branding per institution
  • School-specific public website pages
📚
Virtual Classrooms & LMS
A fully-featured Learning Management System that brings teaching and learning online with rich classroom tools and content delivery.
  • Create & manage virtual classrooms
  • Upload learning materials & resources
  • Assignments creation & submission portal
  • Student enrollment & attendance tracking
  • Module-level content management
📹
Live Virtual Meetings & Calls
Integrated real-time video call infrastructure with secure, token-authenticated meeting rooms — no third-party app required.
  • Classroom live call sessions (Live / Scheduled / Ended)
  • Media controls: mute, camera, screen share
  • Password-protected meeting links via signed tokens
  • Global virtual meetings management panel
  • Meeting status pill indicators
📊
Results Portal & Performance Analytics
Give students, parents, and staff a transparent window into academic performance with rich reporting dashboards and GPA tracking.
  • Term-by-term result cards
  • GPA summary bar (Good / OK / Poor thresholds)
  • Per-subject performance charts
  • Lecturer mark-entry interface
  • Downloadable academic reports
🔔
Announcements & Notifications
Keep everyone in the loop with a powerful, targeted notification system that reaches every role across the platform in real time.
  • School-wide & class-level announcements
  • Alerts for application status changes
  • Notification bell with unread count
  • Admin broadcast panel
  • Role-filtered news feeds
🔐
Enterprise-Grade Security & Auth
Bank-level authentication architecture ensuring every user, session, and data exchange is protected end to end.
  • Secure password hashing (Werkzeug)
  • Google OAuth signup / completion flow
  • Signed token URLs (itsdangerous)
  • Forgot / reset password via email link
  • Role-based access control on every route
💬
Contact & Communication Hub
A centralised inbox for inbound enquiries, plus broadcast communication tools for admins to reach students, parents, and staff.
  • Public contact form with SMTP email delivery
  • Admin inbox with message management
  • Email confirmations on key actions
  • Message status tracking (read / unread)
  • Staff directory integration
