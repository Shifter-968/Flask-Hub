# School Hub Competitive Roadmap

Target: reach and surpass Shunifu-level capabilities with better educational outcomes and operational transparency.

## Now Implemented (Phase 1)
- Finance operations foundation:
  - School admin finance module: account creation, charges, payments, adjustments, reminders.
  - Parent fee visibility: linked account balances and recent ledger entries in parent dashboard.
  - New schema: schema_finance_ops.sql.

## Phase 2 (Next 2-4 weeks)
- Payment rails:
  - Add live payment handlers for MTN MoMo, Stripe, and bank transfer verification.
  - Auto-post successful transactions into student_finance_transactions.
- Fee policy engine:
  - Configure fee structures per class/programme/term.
  - Auto-generate recurring charges.
- Parent trust UX:
  - Downloadable statements, receipt timeline, dispute comments.

## Phase 3 (4-8 weeks)
- Attendance and risk engine:
  - Daily attendance register (not only virtual calls).
  - At-risk scoring model using attendance + grades + assignment completion.
  - Intervention workflow and follow-up tracking.
- Timetable intelligence:
  - Constraint-based clash detection and optimization suggestions.

## Phase 4 (8-14 weeks)
- Learning system depth:
  - LMS authoring enhancements (module sequencing, graded quizzes, certificates).
  - SCORM package ingestion pipeline and tracking.
  - Teacher AI lesson planner with HOD approval and audit logs.

## Phase 5 (14-20 weeks)
- Advanced institutional modules:
  - Teacher CPD and micro-credential tracking.
  - Work-based learning and internship placement logs.
  - Scholarship application and review workflow.
  - Exam-board export and tamper-evident result verification.

## Competitive Differentiators (Outdo Strategy)
- Outcomes, not features:
  - Track and display reduction in fee disputes, improved pass rates, and early intervention success rates.
- Explainable AI:
  - AI recommendations with confidence, rationale, and teacher override trails.
- Trust center:
  - Security/compliance dashboard, role-matrix preview, data export self-service.

## KPIs to monitor
- Report card generation time.
- Fee collection improvement rate.
- Parent portal monthly active users.
- At-risk detection lead time.
- Teacher admin hours saved per week.
